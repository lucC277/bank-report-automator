# 🏦 Bank Report Automator

> Python automation that reads financial transactions from a CSV
> file and prints a formatted bank statement with category summary.

## 📋 Features
- Reads CSV with multiple transaction types
- Prints aligned formatted statement in terminal
- Calculates total income, expenses and monthly balance
- Groups and ranks expenses by category with visual bar

## 🛠️ Technologies
- Python 3.12+
- csv (built-in)

## ▶️ How to run
```bash
# Clone the repository
git clone https://github.com/lucC277/bank-report-automator.git
cd bank-report-automator

# Run the script
python extrato.py
```
